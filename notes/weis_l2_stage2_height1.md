# 段階付き \(ba^*b\) 対カウント言語(「Weis L2」族)・ステージ 2 は一般化スター高さ 1

作業日: 2026-07-22。検証スクリプト: `scripts/weis_l2_family.py`(標準ライブラリのみ、決定的)。

## 0. 位置づけ

CANDIDATES.md Tier 4 の「Weis L2(位数 48 の群; 非有界パターン `ba*b` の
セグメント段階カウント)」に対応する言語族への攻略記録。Weis (2011) の
原文定義の監査は本セッションの環境から不可能だった(§6)。そのため本ノートは
**記録された記述(「`ba*b` の段階カウント」)に合致する言語族全体**を precisely
定義し、そのステージ 2 の場合を完全に解決する。

## 1. 定義

\(w \in \{a,b\}^*\)。位置の**位相** = その位置より前の \(a\) の個数 mod \(m\)。
パターン \(b\,a^j\,b\)(\(j \ge 0\))の出現は「連続する \(b\) の対」と 1 対 1。

- **段階対カウント**
  \(N_{p,q}(w) = \#\{\text{連続 } b \text{ 対}: 先頭 b \text{ の位相} = p,\
  \text{後続 } b \text{ の位相} = q\}\)(\(q - p \equiv j \pmod m\))。
- **セグメント剰余カウント**(Thomas 型・内部ラン版)
  \(T'_s(w) = \#\{b \text{ で終端される } a\text{-ラン}: 長さ \equiv s \pmod d\}\)
  (語頭ランを含む。\(T' := T'_1\), \(d = 2\))。

**ステージ 2 の Weis-L2 族** := \(\{N_{p,q} \equiv h \pmod k\}\)
(\(p,q \in \mathbb{Z}_2\), \(k \in \{2,4\}\))と \(\{T' \equiv h \pmod 4\}\) の
Boolean 閉包。

## 2. 定理(機械検証済み)

**ステージ 2 の族のすべての言語は一般化スター高さ \(\le 1\)。**

検証は 3 層(RESULTS.md §5 の標準に準拠):

1. **厳密証明**(積オートマトン全探索、サンプリングなし):
   各原子値が下記認証済み特徴状態の関数であること
   (`scripts/weis_l2_family.py` [3][4])。
2. **独立 end-to-end**: セル表 vs 定義直接カウント、長さ ≤ 300 の
   ランダム 3 万語で 27 万比較一致([5])。
3. **明示式の独立検証**: \(T' \equiv 0 \pmod 4\) の完全な高さ 1 式を
   `re` エンジンで評価、全語(長さ ≤ 13)+ランダム 2 万語で一致([6])。

## 3. 鍵 1: 有限ラン符号のフラット計数(§6.3 の障害の解消)

RESULTS.md §6.3 が記録した障害:「トークン化が位相追跡を壊す」。すなわち
偶数ラン \((aa)^*\) を内部に含むトークン言語は構文モノイドに \(\mathbb{Z}/2\)
を含み星無しでない。よってブロック/イベント区切りのトークン化はすべて失敗する。

**解消**: トークンを**有限**接頭符号
\[
X = \{\,b,\ aa,\ ab\,\}
\]
に切る。全トークンが長さ ≤ 2 なので自明に星無し。任意の語は一意に
\(w = x_1 \cdots x_n \cdot \rho\)(\(x_i \in X\), 尾部 \(\rho \in \{\varepsilon, a\}\))
と分解する。パリティは「パース整列」に自動的に載る: ラン \(a^{2t}b\) は
\((aa)^t \cdot b\)、ラン \(a^{2t+1}b\) は \((aa)^t \cdot ab\) とパースされ、
**奇数ランがちょうど 1 個の \(ab\) トークンを生む**。

**厳密な整数恒等式**(全語長 ≤ 16 で全数検証済み):
\[
\#ab \;=\; |w|_a + 2|w|_b - 2n - [\rho = a], \qquad \#ab = T'.
\]
係数 2 のおかげで \(T' \bmod 4\) は
\((|w|_a \bmod 4,\ |w|_b \bmod 2,\ n \bmod 2,\ \rho)\) だけで決まる。
各特徴は高さ ≤ 1:
\[
((b^*a)^4)^*(b^*a)^\alpha b^*,\qquad
((a^*b)^2)^*(a^*b)^\beta a^*,\qquad
(X^2)^*X^\nu\rho .
\]
星はいずれも星無し言語の上の 1 段のみ。\(X\) が接頭符号なので \((X^2)^*X^\nu\rho\)
の分解は一意であり計数は well-defined。

明示式の例:
\[
\{T' \equiv t \ (4)\} \;=\;
\bigcup_{\substack{\alpha + 2\beta - 2\nu - r \equiv t\ (4)}}
\Big[((b^*a)^4)^*(b^*a)^\alpha b^*\Big] \cap
\Big[((a^*b)^2)^*(a^*b)^\beta a^*\Big] \cap
\Big[(X^2)^*X^\nu a^r\Big].
\]

## 4. 鍵 2: 位相ウォーク交代(対原子の還元)

\(b\) の位相列 \(p_1,\dots,p_B \in \mathbb{Z}_2\) は奇数ランでのみ反転する。
ゆえに**奇数ランの開始位相は厳密に交互**であり、
\(R_{1,p}\)(位相 \(p\) から始まる奇数ラン数)は
\(\lceil T/2 \rceil\) か \(\lfloor T/2 \rfloor\)(初期位相で決まる)。
行和・列和(W 原子 \(N_r^{(2)}\)、§5 型)と境界旗を併せ、全成分
\(N_{p,q} \bmod 2, 4\) が
\[
\text{LC}(8,4),\ \text{旗},\ \#\text{tok}_X \bmod 4,\ N_0^{(2)} \bmod 4,\ N_1^{(2)} \bmod 4
\]
の関数であることを**積オートマトンで厳密証明**した(手計算の還元は設計指針で
あり、証明は全探索)。ステージ 2 の W 原子(mod 4)自体も §5 の
TokO 構成で同スクリプト内で厳密検証済み([3])。

## 5. ステージ 3 の障害(未解決; 正確な記録)

- 単一ランを被覆する接頭符号の**整数性の束縛**: ラン \(j \equiv r \pmod 3\) の
  消費トークン数は \((j + c_r)/3\), \(c_r \equiv -r \pmod 3\) が強制され、
  フラット計数から得られる線形結合は \((2T_1 + T_2)\) 型 mod 3 に固定される。
  実際 \(X_3 = \{b, aaa, ab, aab\}\) と LC の偶奇で \(T_1 \bmod 2\) は
  **厳密に得られる**(積オートマトン証明)。
- \(T_2 \bmod 2\) には \((T_1 + T_2)\) 型結合(\(c = (5,1)\))が必要だが、
  これは単独トークン \(a\) を要求し充填子 \(aaa\) と接頭衝突する。
- カスケード符号 \(Z_3 = \{b, aaa, aab, aba, abb\}\)(\(r=1\) ランが次の文字を
  1 個奪う)は残差を非線形化し、積オートマトン判定で **不十分**
  (`NOT a function`)。\(Z_{3b}\), 逆向き符号(反転特徴)も同様。
- 逆向きパースは同じ剰余分割を与えるため新しい結合を生まない
  (\(j\) は反転不変)。
- 結論: ステージ 3 の対原子 \(N^{(3)}_{p,q}\) は現有の認証済み特徴族
  (LC・旗・有限符号フラット計数・W 原子 mod 6)の関数ではない。
  PROOF_OBLIGATIONS.md N-L2-M3-001 に登録。

## 6. Weis (2011) の定義監査の状況

- 本セッションの実行環境のネットワークポリシーでは
  scholarworks.umass.edu(学位論文本体)、arxiv.org、irif.fr、
  web.archive.org のいずれにも到達できず、**原文定義の監査は実行不能**
  (AGENTS.md 必須ワークフロー 6 の「不可能な場合は明示」に従い記録)。
- 二次情報(UMass CMPSCI Theory Seminar 2009-11-03 の講演要旨
  "The Power of One Star", P. Weis)によれば、Weis は 4 つの候補言語を
  提示し**そのすべてが一般化スター高さ 1 であることを代数的言語理論の
  道具で示した**とされる。すなわち L2 は(Weis 自身により)反例候補では
  なく解決済みの可能性が高い。原文入手後の照合が必要。
- **位数 48 の群は再現されず**: 記述に合致する自然な族
  (ラン剰余 A 族、Thomas 型 T 族、段階対 C 族、混合 CD/BD/PD 族、
  結合系; \(m \le 6\), \(k \le 8\))の構文モノイドと極大部分群を全走査したが
  現れた位数は 2,3,4,5,6,8,16,18,24,32,64,81,150,192,384 で 48 は無い。
  CANDIDATES.md の「order-48 group」の出典確認も監査項目に含める。

## 7. 帰結

- 「Weis L2」が記録どおり **ステージ 2 の段階付き `ba*b` セグメント
  カウント言語(の Boolean 結合)であるならば、その一般化スター高さは
  1 以下である**(本定理)。構文モノイドが非自明群を含む(§5 の走査で
  部分群 ≦ 32 を確認)ため星無しではなく、高さはちょうど 1。
- ステージ 3 が必要な読み方に対しては、障害が §5 の形で正確に特定された
  未解決問題として残る。
- 手法上の収穫: **有限ラン符号のフラット計数**は、§5.5 の
  パターン条件付きカット+反転でも届かなかった「無限符号の位相追跡」
  障害を、符号の有限性+算術恒等式で回避する新しい認証済み特徴型であり、
  Thomas (1981) のセグメント定理の自己完結な機械証明を副産物として与える
  (\(d = 2\) の場合; CITED から COMPUTED への格上げ)。
